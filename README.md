## indirect object pronouns

| singular | plural |
| -------- | ------ |
| me       | nos    |
| te       | os     |
| le       | les    |

Ejemplo: Yo veo a Maria. Yo la veo.

## direct object pronouns

| --- |
| --- |
| lo  |
| los |
| la  |
| las |

Ejemplo: Compro las flores. Las compro.
